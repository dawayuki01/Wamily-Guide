## 3. コンセプトページ — 大幅リニューアル

### 背景
既存のコンセプトページが情報サイト的で無機質。ガイドブック・各国ページと比べてクオリティに差があった。

### 対応
写真エッセイ形式に全面リニューアル。`css/concept.css` を新規作成し、`concept/index.html` を完全書き直し。

### ページ構成（最終）

| セクション | 内容 |
|---|---|
| HERO | ストックホルムの石畳写真 + 「子どもと行く旅は、もっと、面白い。」 |
| OPENING | 4行改行テキスト（旅が変わった） |
| FULL-BLEED | ロンドンのバスで眠る兄弟 + キャプション |
| 01 きっかけ | 妻の海外挑戦 → 子どもと現地で過ごす体験 |
| 02 旅のかたち | ハワイの秘境ビーチ、友人家族の温かさ |
| PHOTO GRID | 5枚の旅写真（非対称グリッド） |
| PULLQUOTE | 「子どもを連れていくと、その街の違う顔が見える気がする。」 |
| 03 Wamilyとは | 行きたい国に、家族で行けるように。 |
| PROFILE | サワディープロフィール + 訪問国チップ |
| CLOSING | 「家族の数だけ、旅がある。」 |

### デザインのポイント
- 透明ナビ → スクロールで白（`.gnav.scrolled`）
- `--dark-forest: #1a3530`（ディープティールグリーン）をダークパネルに使用
- `IntersectionObserver` によるスクロールアニメーション（`.cp-anim`）
- `clamp()` でレスポンシブタイポグラフィ
- CSS Grid で写真グリッド（非対称5枠）

### Round 2（ピボット対応）

|---|---|---|

### Round 3（海外体験・リアルを強化）
- **オープニング改行**：「荷物は増えた。計画は狂う。思い通りにいかないことばかり。」を1文ずつ改行
- **01 きっかけ 書き直し**：「留守番」エピソード削除 → 海外でのリアルな体験にフォーカス

**01 きっかけ 最終テキスト：**
```
妻が向こうで仕事をしている間、僕が子どもたちと一緒にその国で過ごすこともある。
旅行ではあるんだけど、ちょっとした日常を送る感じで。

言葉が通じない。文化が違う。日本だったら何でもない場面でも、とにかく困る。
子ども2人を連れているから、その大変さが倍になる。

でも、それが全部ひっくるめて、家族として特別な体験になる。
「大変だから行かない」は、僕たちじゃない。
大変なほど、絆が深まって、思い出が強くなる。だって、海外って面白いから。

Wamilyは、そこから始まりました。
```

## 5. 各国ガイドブックページ — 回遊導線修正

### 問題
- ヘッダーのパンくず「ガイドブック」がテキストのみでリンクなし
- フッター「← ガイドブックに戻る」が `../`（トップページ）に遷移していた

### 対応
全10カ国ページを一括修正：

| 場所 | Before | After |
|---|---|---|
| ヘッダーパンくず | `ガイドブック ›`（テキスト） | `[ガイドブック] ›`（`../guidebook/` へのリンク） |
| フッター | `← ガイドブックに戻る` → `../` | `← ガイドブックに戻る` → `../guidebook/` |

**対象ページ：** ロンドン / 台湾 / パリ / ストックホルム / シンガポール / バンコク / マニラ / LA / ハワイ / ソウル

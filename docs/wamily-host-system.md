# Wamily ホストシステム 設計記録

作成日: 2026年4月2日

---

## コンセプト

Wamilyには現地在住の日本人家族「Wamilyホスト」が存在する。
旅する家族とホストを繋ぐことで、情報だけでは届かない「人の温かさ」を提供する。

> 「AIが動いているのに、人の温度がある」—— これがWamilyの核心

---

## MVP設計（Phase 1）

### サイト側の表示ルール

| 条件 | 表示 | CTA |
|---|---|---|
| ホストあり国 | 「〇〇には素敵なWamilyホストがいます」 | Googleフォーム（問い合わせ） |
| ホストなし国 | 「〇〇に住む日本人家族の方へ」 | Googleフォーム（ホスト応募） |

### 問い合わせフロー（ホストあり国）

```
訪問者がフォーム送信
  ↓
【即時】定型文で自動返信（Apps Script）
  ↓
【即時】サワディーにメール通知 + 問い合わせ内容
  ↓
【48h後】未対応なら自動リマインド
  ↓
サワディーがLINEでホストと連携 → 個別対応
  ↓
料金・詳細は個別メッセージで完結
```

### 応募フロー（ホストなし国）

```
現地在住者がフォーム送信
  ↓
【即時】定型文で自動返信（感謝 + 次のステップ案内）
  ↓
【即時】サワディーにメール通知
  ↓
サワディーが審査・連絡 → ホスト登録
```

### AI自動返信の方針

| フェーズ | 対応 | 実装 |
|---|---|---|
| Phase 1（現在） | 定型文テンプレ | Googleフォーム + Apps Script |
| Phase 2（将来） | 内容を読んで返答 | Claude API 呼び出し |
| Phase 3（将来） | 自律的な案件管理 | AIエージェント |

**判断基準:** データと事例が蓄積されてから自動化を深める。信頼性のないAI対応より、温度のある人の対応を優先。

### Apps Script 設定（問い合わせ用）

```javascript
// フォーム送信トリガーで実行
function onFormSubmit(e) {
  const email = e.values[/* メールアドレス列 */];
  const content = e.values.join('\n');

  // 1. 訪問者への自動返信
  GmailApp.sendEmail(email,
    '【Wamily】お問い合わせありがとうございます',
    REPLY_TEMPLATE
  );

  // 2. サワディーへの通知
  GmailApp.sendEmail(OWNER_EMAIL,
    '【Wamily】新規ホスト問い合わせ',
    content
  );

  // 3. 48h後リマインド（ScriptProperties + time-based trigger）
  setReminder(email, content);
}
```

## 将来のビジョン

### AIチーム構想

Wamilyの自動化をチームとして設計する。各チームはAIエージェントが担当し、人（サワディー）が最終判断・ブランドの温度を担う。

#### 編集チーム（稼働中）
- 各国イベント情報の収集・生成（`fetch-events.js`）
- スポット情報の更新・品質チェック（`check-spots.js`）
- フィード投稿の下書き生成（`generate-guide.js`）

#### コミュニティチーム（構築中）
- **ホスト管理エージェント**: 応募受付・定型返信・ホスト登録管理
- **ユーザー管理エージェント**: 問い合わせ初動対応・未返信フラグ・リマインド

#### ブランドチーム（将来）
- メルマガ文章生成
- SNS投稿キュレーション
- サワディーのトーンを学習して代筆

### アプリ化ロードマップ

```
現在: ガイドブックHP（静的 + 自動化パイプライン）
  ↓
中期: ホストマッチング機能（Webアプリ）
  ↓
将来: Wamilyアプリ（ホスト・旅行者・AIチームが統合）
```

## ホスト紹介文の方針（2026年4月2日 追記）

カードのデザイン方針が確定した。

**「ホストのスペック説明より、サワディーがそのホストを温かく紹介するスタンス」**

> ホストの詳細（名前・顔・料金）は後から追加するが、
> 今は「サワディーがそのホストとの体験を持っていて、
> 自信を持って紹介している」という文脈が伝われば十分。

### 今後追加する紹介文のイメージ（ロンドン）
```
「ロンドンのホストとは、子どもが高熱を出した夜のことが忘れられない。
深夜に連絡したのに、すぐに動いてくれた。あの安心感は本物でした。」
— サワディー
```

各ホストとの具体的なエピソードをサワディーから共有してもらい次第、
各国ホストカードに `ed-voice-quote` 風の引用ブロックとして追加予定。
